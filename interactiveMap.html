Interactive Map

What is this map?

This interactive map was my scholarship project, carried out in 2015/2016, supervised by University of Canterbury Adjunct Fellow and Statistics New Zealand Senior Researcher, Richard Penny, funded by Statistics New Zealand. The purpose of this project was to build an interactive map that shows how people commuted to work on the censused day in New Zealand.
Key features:

  1. Source data was retrieved from Statistics New Zealand website.
  2. Geospatial techniques involved
  3. Data cleansing
  4. Exploratory Data Analysis
  5. Data modelling
  6. One-mean table and map
  7. Two-mean table and map
  8. Interactive Scatterplot
