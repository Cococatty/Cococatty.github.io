Welcome to Neptune X

What is Neptune X?

Neptune X is a financial application developed by Carina. It aims to use provide users a good understanding of their spending habits, advice on how they can shape their spendings towards a more desired trend, via the power of Statistics and Computer Science in an interactive and fun manner.
Current key features:

  1. Raw data input
      2. Raw data from different bank statements are consolidated into relational tables, stored securely (encryption and credential required).
      Keyword classification is being applied. There is a table that stores all keywords by account categories specified by users. The application would learn the keywords and automatically classify the spending entry into corresponding category for further analyse.
  2. Cleanse and format data by the source types (Direct Debit, Credit Card, Loan account etc)
  3. Apply various statistical analyses
      4. Exploratory Data Analyse, to generate an overview of the spending.
      Regression Modelling - build liner models by accounts;
      Time Series - analysis the spending characteristics and the time it occurred;
      Bayesian Inference - estimate probabilities along with the amount of spending by categories.
  4. Present answers to users' questions selected on application, by using simple English text and visualization, in an user-friendly but informative manner.
