Interactive Map Scholarship Project

What is this map?

This interactive map was part of the deliveries of my scholarship project, carried out from November 2015 to February 2016, supervised by University of Canterbury Adjunct Fellow and Statistics New Zealand Senior Researcher, Richard Penny, funded by Statistics New Zealand. The purpose of this project was to build an interactive map that shows how people commuted to work on the censused day in New Zealand.

Key features:

  1. Source data was retrieved from Statistics New Zealand website.
  2. Geospatial techniques involved
  3. Data cleansing
  4. Exploratory Data Analysis
  5. Data modelling
  6. One-mean table and map
  7. Two-mean table and map
  8. Interactive Scatterplot

The other deliveries items were a project report and presentation to Statistics New Zealand. Statistics New Zealand has then further developed this topic.

Gain from this project:

  • I learnt how to build a Shiny application during the project by myself.
  • A highly open project with guideline and inspiration provided by top expert, Richard.
